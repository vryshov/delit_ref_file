Описание
Программа на питоне = модуль "mod_DelLitRefFile.py"
Программа удаляет в текстовом файле источнике "my_file1.txt" все вхождения ссылок на литературу,
а результат записывает в новый файл "my_file2.txt".
Ссылки представляют собой текстовые блоки в квадратных скобках, например: [1], [25], [635].
Обработка файла производится по строкам. На самом деле строки - это абзацы текста, завершающиеся символом перевода строки \n.
***

Как пользоваться программой:

Первый вариант
1) Помещаем в файл "my_file1.txt" текст книги для обработки рядом с файлом-модулем "mod_DelLitRefFile.py" на питоне.
2) Кликаем в этот файл-модуль - откроется окно редактора Sablime с текстом программы модуля (если что-то нужно - правим).
3) Запускаем на исполнение этот модуль - Ctrl+B
4) Смотрим результат в новом файле "my_file2.txt".

Второй вариант
(с интерпретатором питона "IDLE (Python 3.5 32-bit)")
1) Запускается интерпретатор питона.
- Появаляется окно "Python 3.5.2 Shell"

2) Загруажется модуль питона "mod_DelLitRefFile.py"
- В меню в опции "File" кликаем "Open"
- В окне "Открыть" в каталоге "_Удаление []" (открываем этот каталог) - указываем "mod_DelLitRefFile.py".
- открывается окно программы "IDLE".
- Нажимаем клавишу F5 (или в меню в опции "Run" кликаем "Run Module")
- В интерпретаторе питона увидем сообщение:
== RESTART: C:\Python\Python35-32\_Удаление []\mod_DelLitRefFile.py ==

3) Записываем в каталоге "C:\Python\Python35-32\_Удаление []" конверируемый текстовый файл под имененм "my_file1.txt".

4) В интерпретаторе питона после приглашения в командной строке даем команду в виде функции:
>>> DelLitRefFile()

5) После работы программы модуля "mod_DelLitRefFile.py"
появится файл-результат - "my_file2.txt"
***

 Замечание 1:
Можно использовать функцию: DelLitRefFile_(<число от 1 до 5 (или более)>),
где явно указывается максимальная длина удаляемых блоков [1...n].

 Замечание 2:
В файле-результате - "my_file2.txt" в конце дается информация:
Пример.

 Время: 9.81 сек.
=
